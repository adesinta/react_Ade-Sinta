1. Playground tanpa menggunakan template
   - Input user : Bagaimana caranya menggunakan flowbite di vite?
   - Output assistant :
		Untuk menggunakan Flowbite di Vite, ikuti langkah-langkah berikut:
		a. Instalasi Flowbite:
		- Buka terminal dan jalankan perintah `npm install flowbite` atau `yarn add flowbite` untuk menginstal Flowbite ke dalam proyek Vite Anda.
		- Tunggu hingga proses instalasi selesai.

		b. Impor Flowbite ke dalam proyek Anda:
		- Buka file `main.js` di direktori proyek Anda.
		- Tambahkan baris berikut di bagian atas file untuk mengimpor Flowbite:
		javascript
		import 'flowbite/dist/flowbite.css';
		
		c. Gunakan komponen Flowbite:
		- Sebagai contoh, kita akan menggunakan komponen tombol Flowbite.
		- Buka file Vue komponen yang ingin Anda gunakan Flowbite.
		- Impor komponen tombol dengan menambahkan baris berikut di bagian atas file:
			javascript
			import { Button } from 'flowbite';

		- Tambahkan
